Skill Introduction

System Query Language or SQL is a programming language allows users to interact with relational databases management systems (RDBMS) specifically to manage data within a database, performing operations such as storing, retrieving data and or modifying data within a database.

Some History

In the 1970's Dr. Codd published a research paper to the Association of Computer Machinery, to which he designed a model for relational database management systems (RDBMS). IBM then used Dr. Codd's model to develop SQL, which is now the widely accepted language for RDBMS.

Importance

SQL is important because it is the standard language for interacting and managing data within a database. This allows for users to efficiently and effectively store, retrieve, update and or delete data which is essential for handling massive amounts of data. SQL allows for data to be organized in any form, accesible and easy to analyze which is effective for software development.
