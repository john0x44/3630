Contact Management System

We will be creating a contact management system utilizing SQL, it will store and manage contact details, the database will have a contacts table to which will include a name,phone,email, and address, it will as well have a primary key called contact id. We will add a groups table to organize our contacts into different groups such as friends,family or work. Each contact will belong to one group this is called a one to many relationship. We can then utilize this created database to integrate it into a python application in which the end user can manage their contacts data easily and efficiently.
